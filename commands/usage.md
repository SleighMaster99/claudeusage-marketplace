# Usage 조회

## 지시사항

1. 다음 Bash 명령어를 실행하여 현재 사용량을 조회하세요:
   ```bash
   node ${CLAUDE_PLUGIN_ROOT}/dist/statusline.js
   ```

2. 명령어 결과를 사용자에게 보기 좋게 표시하세요
3. 추가로 다음 정보를 안내하세요:
   - 세션 사용량: 5시간 주기로 초기화
   - 주간 사용량: 7일 주기로 초기화
   - 80% 이상 시 경고 표시

- 인터넷 연결 확인
